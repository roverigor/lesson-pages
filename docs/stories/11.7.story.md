## Descrição

Como **coordenador**, quero ver quantas mensagens cada aluno enviou no chat das aulas Zoom — capturadas automaticamente via cron noturno — para incluir no ranking de engajamento.

**Desbloqueado:** `report:read:admin` já ativo no app Zoom. Endpoint: `GET /report/meetings/{meetingId}/chat`.

---

## Acceptance Criteria

- [ ] **AC1 — Tabela:** `zoom_chat_messages(id, zoom_meeting_id, sender_name, student_id, sent_at, message_id UNIQUE)`
- [ ] **AC2 — Tabela ranking:** `engagement_daily_ranking(student_id, cohort_id, ref_date, wa_messages, zoom_chat_messages, attendance_count, engagement_score)`
- [ ] **AC3 — Action no zoom-attendance:** `import_meeting_chat` busca chat de uma reunião e insere em `zoom_chat_messages` (deduplicado)
- [ ] **AC4 — Cron noturno (02h AM):** pg_cron chama `zoom-attendance?action=nightly_engagement_sync`
  - Busca meetings do dia anterior (chat_imported=false)
  - Para cada: importa chat via `import_meeting_chat`
  - Recalcula `engagement_daily_ranking` para cohorts afetados
- [ ] **AC5 — Fuzzy match:** sender_name → student_id usando ilike match por primeiro nome
