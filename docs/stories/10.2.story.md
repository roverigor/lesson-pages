## Descrição

Como **aluno da Academia Lendária**, quero ver um resumo dos pontos principais de cada aula logo abaixo do vídeo — gerado automaticamente — para que eu possa revisar o conteúdo rapidamente sem precisar assistir a gravação completa.

**Contexto técnico:** O Zoom pode gerar transcrição VTT automaticamente (se `recording.completed` incluir `transcript_download_url`). O webhook de `recording.completed` já será capturado na Story 10.1. Esta story adiciona: (1) download do transcript VTT, (2) chamada ao Anthropic API (claude-haiku-4-5) para gerar resumo estruturado, (3) salvar em `class_recordings.summary` e `class_recordings.transcript_text`.

---

## Acceptance Criteria

- [x] **AC1 — Transcript baixado:** quando `recording.completed` contém VTT, o texto é baixado e salvo em `class_recordings.transcript_text`
- [x] **AC2 — Resumo gerado:** Anthropic API (claude-haiku-4-5) gera um resumo de 3-5 bullet points dos temas abordados, salvo em `class_recordings.summary`
- [x] **AC3 — Graceful degradation:** se não houver transcript disponível, `summary` fica NULL e a UI exibe "Resumo indisponível para esta aula"
- [x] **AC4 — Template do resumo:** formato fixo:
  ```
  ## Resumo da Aula — {título}
  
  **Temas abordados:**
  • Tema 1
  • Tema 2
  • Tema 3
  
  **Próximos passos mencionados:**
  • ...
  ```
- [x] **AC5 — Exibição na UI:** `/aulas` exibe o summary abaixo do vídeo, com formatação markdown simples
- [x] **AC6 — Sem custo em aulas sem transcript:** a chamada ao Anthropic só é feita quando há transcript disponível

## Risk Notes

- **Risk:** Transcrição Zoom em inglês mesmo que a aula seja em português
- **Mitigação:** Prompt instrui o modelo a resumir em português independente do idioma do transcript

- **Risk:** `ANTHROPIC_API_KEY` não configurada
- **Mitigação:** Verificar antes de implementar; usar `ANTHROPIC_API_KEY` (já pode existir no projeto)
