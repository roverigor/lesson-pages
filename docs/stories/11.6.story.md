## Descrição

Como **coordenador**, quero ver quantas vezes cada aluno enviou mensagem no grupo WhatsApp da turma — capturado automaticamente via webhook Evolution API — para incluir participação no ranking de engajamento.

---

## Acceptance Criteria

- [ ] **AC1 — Tabela:** `whatsapp_group_messages(id, group_jid, sender_phone, student_id, cohort_id, sent_at, message_type, evolution_message_id UNIQUE)`
- [ ] **AC2 — Captura tempo real:** `delivery-webhook` processa `messages.upsert` para JIDs terminando em `@g.us`
- [ ] **AC3 — Deduplicação:** INSERT com `ON CONFLICT (evolution_message_id) DO NOTHING`
- [ ] **AC4 — Mapeamento student_id:** lookup `students.phone → students.id` (normalizado: apenas dígitos)
- [ ] **AC5 — Mapeamento cohort_id:** lookup `cohorts.whatsapp_group_jid = group_jid` (coluna existe ou adicionar)
- [ ] **AC6 — Mensagens próprias ignoradas:** `fromMe = true` → skip
- [ ] **AC7 — Contagem visível:** ranking no hub da turma mostra dias ativos WA por aluno
