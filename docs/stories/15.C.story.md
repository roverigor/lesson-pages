## O que fazer

Implementar callback outbound para ActiveCampaign confirmando dispatch concluído. Após `dispatch-survey` enviar form com sucesso, chama `ac-report-dispatch` que escreve `custom_field` no contato AC + persiste em `ac_dispatch_callbacks` com retry exponencial 3x.

## Business Value

- **Fecha o loop AC ↔ nosso sistema** (FR-11) — AC sabe que disparo aconteceu para automações posteriores
- **Resiliência via retry** (NFR-5) — falhas transientes não perdem confirmação
- **Audit completo** em `ac_dispatch_callbacks` — debug + reprocessamento manual
- **Pequeno mas crítico:** apenas 6h, mas sem isso AC fica sem feedback

## Dev Notes

### Skeleton

```typescript
serve(async (req) => {
  // Auth: service_role only
  const auth = await verifyServiceRole(req);
  if (!auth) return jsonResponse({ error: "unauthorized" }, 401);

  const { link_id, ac_contact_id, custom_field_id, value } = await req.json();

  // Idempotência check
  const { data: existing } = await sb.from("ac_dispatch_callbacks")
    .select("acknowledged_by_ac")
    .eq("link_id", link_id)
    .single();

  if (existing?.acknowledged_by_ac) return jsonResponse({ status: "already_acknowledged" });

  let lastError = "";
  for (let attempt = 1; attempt <= 3; attempt++) {
    try {
      const res = await fetch(`${AC_API_URL}/api/3/contacts/${ac_contact_id}/fieldValues`, {
        method: "PUT",
        headers: { "Api-Token": AC_API_KEY, "Content-Type": "application/json" },
        body: JSON.stringify({ fieldValue: { contact: ac_contact_id, field: custom_field_id, value } })
      });

      if (res.ok) {
        await sb.from("ac_dispatch_callbacks").upsert({
          link_id, ac_contact_id, status: "ok",
          retries: attempt, last_attempt_at: new Date().toISOString(),
          acknowledged_by_ac: true
        });
        return jsonResponse({ status: "ok" });
      }

      if (res.status >= 400 && res.status < 500) {
        // Não-retry em 4xx
        lastError = `4xx ${res.status}`;
        break;
      }
      lastError = `5xx ${res.status}`;
    } catch (e) {
      lastError = e.message;
    }
    if (attempt < 3) await sleep(Math.pow(4, attempt - 1) * 1000); // 1s, 4s, 16s
  }

  // Falha total
  await sb.from("ac_dispatch_callbacks").upsert({
    link_id, ac_contact_id, status: "failed",
    retries: 3, last_attempt_at: new Date().toISOString(),
    error_message: lastError
  });
  await alertSlack(`AC callback failed link_id=${link_id}: ${lastError}`);
  return jsonResponse({ status: "failed", error: lastError }, 502);
});
```
