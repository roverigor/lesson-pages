## Descrição

Como **coordenador**, quero poder importar transcrições de gravações Zoom que já existem (anteriores ao webhook) e gerar resumos AI para elas — para que todas as aulas tenham resumo disponível no painel, não apenas as futuras.

### Contexto Técnico

- O webhook `recording.completed` já baixa VTT e gera resumo via GPT-4o-mini
- Funciona apenas para gravações NOVAS (forward-looking)
- Gravações históricas em `class_recordings` que já têm `video_url` mas não têm `transcript_text` nem `summary`
- A Zoom Cloud Recording API permite baixar transcrições de gravações existentes usando o download_url + access_token
- O token Server-to-Server OAuth já está configurado

---

## Acceptance Criteria

- [ ] **AC1 — Action batch_import_transcripts:** Nova action em `zoom-attendance` ou `zoom-webhook` que processa gravações sem transcrição
- [ ] **AC2 — Busca em class_recordings:** Seleciona registros onde `transcript_text IS NULL` e `video_url IS NOT NULL`
- [ ] **AC3 — Download VTT da Zoom API:** Para cada gravação, busca a transcrição via Zoom API (GET /meetings/{meetingId}/recordings)
- [ ] **AC4 — Geração de resumo AI:** Reutiliza `generateAndSaveSummary()` existente para gerar resumo a partir da transcrição
- [ ] **AC5 — Progresso incremental:** Processa em batches de 5 gravações por execução para respeitar rate limits da Zoom API
- [ ] **AC6 — Botão no admin:** Botão "Importar Transcrições Pendentes" na tab Gravações que chama a action
- [ ] **AC7 — Log de resultado:** Retorna contagem de transcrições importadas, resumos gerados, erros
