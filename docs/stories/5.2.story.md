## Descrição

Como **admin da Academia Lendária**, quero poder cadastrar os nomes alternativos que cada mentor usa no Zoom diretamente na página `/equipe`, para que o sistema de matching automático os reconheça corretamente nos relatórios importados — sem precisar editar SQL manualmente.

**Contexto técnico:** O campo `mentors.aliases TEXT[]` já existe no banco com GIN index. A função `mark_mentor_participants()` já usa esse campo com 8 regras de matching. Esta story adiciona a UI para gerenciar esses aliases inline nos cards de mentor da página `/equipe/index.html`.

---

## Acceptance Criteria

- [ ] **AC1 — Seção de aliases visível:** cada card de mentor em `/equipe` exibe uma seção "Nomes no Zoom" com os aliases cadastrados como tags removíveis
- [ ] **AC2 — Adicionar alias:** existe um input + botão "Adicionar" que insere novo alias ao array `aliases` do mentor no Supabase; a tag aparece imediatamente após salvar sem reload de página
- [ ] **AC3 — Remover alias:** cada tag tem um botão [×]; ao clicar, o alias é removido do array no Supabase e a tag desaparece imediatamente
- [ ] **AC4 — Validação de tamanho:** alias com menos de 3 caracteres exibe mensagem de erro e não é salvo
- [ ] **AC5 — Validação de unicidade local:** se o alias já existe na lista do mesmo mentor, exibe aviso e não duplica
- [ ] **AC6 — Estado vazio:** quando aliases = [], exibe "(nenhum alias cadastrado)" com call-to-action sutil
- [ ] **AC7 — Feedback visual:** botão "Adicionar" mostra estado de carregamento durante o save; erro de rede é exibido inline

## Technical Notes

```javascript
// Load mentors com aliases (garantir que o campo está no SELECT)
const { data: mentors } = await sb
  .from('mentors')
  .select('id, name, active, aliases, ...')
  .eq('active', true);

// Adicionar alias
async function addAlias(mentorId, currentAliases, newAlias) {
  const updated = [...(currentAliases || []), newAlias.trim()];
  const { error } = await sb
    .from('mentors')
    .update({ aliases: updated })
    .eq('id', mentorId);
  return error;
}

// Remover alias
async function removeAlias(mentorId, currentAliases, aliasToRemove) {
  const updated = currentAliases.filter(a => a !== aliasToRemove);
  const { error } = await sb
    .from('mentors')
    .update({ aliases: updated })
    .eq('id', mentorId);
  return error;
}
```
