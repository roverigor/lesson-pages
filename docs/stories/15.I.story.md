## O que fazer

Implementar webhook receiver para Meta Cloud API `messages.update` events. Recebe delivery + read receipts e atualiza `survey_links.delivered_at` / `read_at` correlacionando via `meta_message_id`.

V1 SEM HMAC X-Hub-Signature-256 (META_APP_SECRET indisponível — decisão user 2026-05-04). Mitigações: META_WEBHOOK_VERIFY_TOKEN para subscription + idempotência + schema validation + rate limit.

## Business Value

- **Tracking 4 timestamps completo** (NFR-16) — delivered + read confirmados pelo Meta
- **NFR-17 implementação** (delivery webhook) — habilita drill-down aluno timeline rica
- **CS visibilidade real** do que aluno fez com a mensagem (entregue? leu? ignorou?)
- **EC-20 race condition resolvido** via UPSERT por meta_message_id

## Dev Notes

### Skeleton

```typescript
const META_WEBHOOK_VERIFY_TOKEN = Deno.env.get("META_WEBHOOK_VERIFY_TOKEN")!;

serve(async (req) => {
  const url = new URL(req.url);

  if (url.searchParams.get("warmup") === "1") return new Response("ok", { status: 200 });

  // GET — subscription verify
  if (req.method === "GET") {
    const mode = url.searchParams.get("hub.mode");
    const token = url.searchParams.get("hub.verify_token");
    const challenge = url.searchParams.get("hub.challenge");

    if (mode === "subscribe" && token === META_WEBHOOK_VERIFY_TOKEN) {
      return new Response(challenge, { status: 200 });
    }
    return new Response("forbidden", { status: 403 });
  }

  if (req.method !== "POST") return jsonResponse({ error: "method_not_allowed" }, 405);

  // Rate limit
  const ip = req.headers.get("x-forwarded-for") || "unknown";
  if (isRateLimited(ip)) return jsonResponse({ error: "rate_limited" }, 429);

  const payload = await req.json();
  const statuses = payload?.entry?.[0]?.changes?.[0]?.value?.statuses || [];

  for (const s of statuses) {
    const { id: metaMessageId, status, timestamp } = s;
    if (!["sent", "delivered", "read"].includes(status)) continue;

    const updateField = status === "delivered" ? "delivered_at" : status === "read" ? "read_at" : null;
    if (!updateField) continue;

    const ts = new Date(parseInt(timestamp) * 1000).toISOString();

    // UPSERT preserving existing values
    const { data, error } = await sb.from("survey_links")
      .update({ [updateField]: ts })
      .eq("meta_message_id", metaMessageId)
      .is(updateField, null) // Só atualiza se ainda nulo (idempotência)
      .select();

    if (!data?.length) {
      console.log({ event: "skip_unknown", meta_message_id: metaMessageId, status });
      continue;
    }
    console.log({ event: "tracked", meta_message_id: metaMessageId, status });
  }

  return jsonResponse({ status: "ok" });
});
```

### Rate Limit In-Memory

```typescript
const limits = new Map<string, { count: number, ts: number }>();
function isRateLimited(ip: string): boolean {
  const now = Date.now();
  const entry = limits.get(ip);
  if (!entry || now - entry.ts > 60_000) {
    limits.set(ip, { count: 1, ts: now });
    return false;
  }
  entry.count++;
  return entry.count > 100;
}
```
