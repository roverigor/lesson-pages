## Descrição

Como **coordenador**, quero que os links de NPS enviados via WhatsApp expirem em 7 dias — para que alunos que clicarem fora do prazo vejam mensagem de encerramento clara, e o sistema não aceite respostas fora do ciclo avaliativo.

**IMPORTANTE:** `survey_links` já existe (EPIC-004) — apenas adicionar coluna `expires_at`. NÃO criar nova tabela.

---

## Acceptance Criteria

- [ ] **AC1 — Migration:** adiciona `expires_at TIMESTAMPTZ` em `survey_links` (nullable — links antigos sem expiração continuam válidos)
- [ ] **AC2 — dispatch-survey:** ao criar survey_link, define `expires_at = NOW() + INTERVAL '7 days'`
- [ ] **AC3 — submit-survey:** ao validar token, verifica se `expires_at IS NOT NULL AND expires_at < NOW()` → retorna `{ error: "token_expired" }`
- [ ] **AC4 — Formulário NPS:** ao receber `token_expired`, exibe tela "Prazo encerrado" sem opção de submit
