## Descrição

Como **coordenador**, quero acessar `/turma` e ver uma lista de turmas; ao clicar em uma turma, ver todos os alunos matriculados com ranking de presença e participação no WhatsApp — em uma única tela — para ter visão operacional completa sem navegar entre 3 painéis.

---

## Acceptance Criteria

- [ ] **AC1 — Lista de turmas:** `/turma/index.html` lista todas as turmas (cohorts) com nome, total de alunos e % presença geral
- [ ] **AC2 — Página da turma:** ao clicar em uma turma, abre `/turma/detalhe.html?id={cohort_id}` com: lista de alunos, % presença individual, ranking WhatsApp (dias ativos), link para perfil 360
- [ ] **AC3 — Ranking de presença:** alunos ordenados por % presença DESC; badge vermelho se < 70%
- [ ] **AC4 — Ranking WhatsApp:** coluna "Dias Ativos WA" a partir de `engagement_daily_ranking`; mostra 0 se sem dados
- [ ] **AC5 — Link para perfil 360:** botão "Ver Perfil" por aluno → `/aluno/perfil.html?id={student_id}`
- [ ] **AC6 — Acesso restrito:** login com senha admin (mesmo padrão do `/presenca`)
