## O que fazer

Implementar tab "Integrações" em `/cs/integrations` com 2 sub-tabs: (1) Mappings AC CRUD — `ac_product_mappings` table; (2) Eventos AC — lista `ac_purchase_events` últimos 30d com filtros, drill-down evento (payload + status + erros), reprocessar evento manual.

## Business Value

- **FR-15 mappings CRUD:** CS configura auto-onboarding sem dev
- **FR-14 dashboard observabilidade:** taxa sucesso/falha visíveis
- **Audit transparency:** rastreio total dos eventos AC
- **Reprocessar manual:** recovery em casos de bug temporário

### Reprocessar Action

```javascript
async function reprocessEvent(eventId) {
  await supabase.from('ac_purchase_events')
    .update({ status: 'received', retry_count: 0, last_error: null, processing_started_at: null, processed_at: null })
    .eq('id', eventId);
  showToast('Evento enfileirado para reprocessar (até 30s)');
}
```
