## Descrição

Como **mentor ou coordenador**, quero ver o horário de início e fim de cada aula nos modais do calendário — ao clicar em um dia — para saber exatamente quando cada sessão acontece sem precisar navegar para outra página.

**Contexto técnico:** O modal de dia em `/calendario/index.html` exibe turma, professor e host, mas não exibe hora. Os campos `start_time` e `end_time` já existem na tabela `classes` e já são carregados na query inicial (`loadEvents()`). Basta incluí-los na renderização do modal.

---

## Acceptance Criteria

- [ ] **AC1 — Horário no modal de dia:** ao clicar em qualquer dia do calendário que tenha sessões, o modal exibe o horário no formato `HH:MM – HH:MM` para cada aula listada
- [ ] **AC2 — Formato 24h:** horário exibido em formato 24h (ex: `19:00 – 21:30`), sem AM/PM
- [ ] **AC3 — Horário nos event pills:** os pills de eventos no grid do calendário também exibem o horário de início abreviado (ex: `19h`) quando houver espaço suficiente (desktop)
- [ ] **AC4 — Fallback gracioso:** se `start_time` ou `end_time` for `null`, não exibir horário (não quebrar o modal)
- [ ] **AC5 — Sem regressão:** demais informações do modal (turma, professor, host, substituto) continuam exibidas corretamente

---

## Tasks

- [ ] Verificar que `start_time` e `end_time` estão incluídos na query de `loadEvents()` (confirmar via `SELECT`)
- [ ] Atualizar função de renderização do modal de dia para incluir horário no formato `HH:MM – HH:MM`
- [ ] Atualizar renderização dos event pills no grid para incluir hora de início abreviada (`19h`)
- [ ] Adicionar fallback para `start_time = null` (não renderizar horário, não quebrar)
- [ ] Testar em diferentes turmas que têm horários diferentes (19h, 21h, etc.)
