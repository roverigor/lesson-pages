## Descrição

Como **aluno recebendo link via WhatsApp**, quero clicar no link e responder o NPS em 1 clique — sem fazer login — e receber feedback claro de que minha resposta foi registrada ou que o prazo encerrou.

O formulário `/avaliacao/responder.html` já existe. Esta story garante que ele lida corretamente com todos os estados do token.

---

## Acceptance Criteria

- [ ] **AC1 — Token válido:** formulário carrega normalmente (comportamento atual)
- [ ] **AC2 — Token expirado:** exibe tela "Prazo encerrado — O link de avaliação expirou após 7 dias."
- [ ] **AC3 — Token já usado:** exibe tela "Avaliação já enviada — Sua resposta já foi registrada. Obrigado!"
- [ ] **AC4 — Token inválido:** exibe tela "Link inválido — Este link de avaliação não é válido."
- [ ] **AC5 — Submit com sucesso:** tela de confirmação "Obrigado! Sua avaliação foi enviada." (já existe)
