## Descrição

Como **coordenador da Academia Lendária**, quero ver um painel que mostre quais aulas ainda estão sem gravação, sem resumo ou sem materiais — de forma consolidada — para poder identificar e resolver lacunas de conteúdo sem precisar checar manualmente cada aula.

**Contexto técnico:** A `student_attendance` tem registro de todas as datas de aula por cohort. `class_recordings` tem as gravações disponíveis. O cruzamento mostra quais datas de aula têm ou não conteúdo disponível.

---

## Acceptance Criteria

- [x] **AC1 — Seletor de turma:** dropdown para selecionar cohort (admin vê todos, mentor vê suas turmas)
- [x] **AC2 — Lista de aulas:** todas as datas de aula do cohort exibidas em ordem, com badges de status para cada componente:
  - 🎬 Gravação: ✅ disponível / ⏳ aguardando / ❌ ausente
  - 📝 Resumo IA: ✅ disponível / ❌ ausente
  - 📎 Materiais: ✅ (N arquivos) / ❌ sem materiais
- [x] **AC3 — Taxa de completude:** exibe "X de Y aulas com gravação — Z%"
- [x] **AC4 — Ação rápida:** botão "Adicionar gravação manualmente" para inserir link de reunião sem webhook
- [x] **AC5 — Acesso restrito:** apenas admin (sessão com role admin) vê este painel; alunos e mentores não têm acesso
