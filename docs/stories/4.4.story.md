# Story 4.4 — Página Pública: /avaliacao/responder.html

**Epic:** EPIC-004  
**Status:** Ready  
**Type:** Frontend / Public  
**Depende de:** Story 4.1, 4.2

## O que fazer

Criar `/avaliacao/responder.html` — página pública onde o aluno responde a survey via link tokenizado, com UX conversacional (uma tela por vez) no estilo Tally mas com o dark premium da Academia Lendária.

## Acceptance Criteria

### Fluxo de telas
- [ ] **Tela 0 (loading):** Valida token via `submit-survey` pré-check ou busca survey. Mostra spinner.
- [ ] **Tela de erro:** Token inválido, expirado ou já respondido → mensagem amigável ("Essa pesquisa já foi respondida ou o link é inválido")
- [ ] **Tela 1 (pergunta principal NPS):** "Como você avalia [pergunta]?" com botões de 0 a 10 em linha. Botões coloridos por faixa (vermelho 0-6, amarelo 7-8, verde 9-10). Legenda "Nada satisfeito" e "Muito satisfeito".
- [ ] **Tela 1 (pergunta principal CSAT):** Pergunta com 5 estrelas interativas (clicar → preenche até a estrela selecionada)
- [ ] **Tela 2 (follow-up, opcional):** Textarea livre "O que motivou sua nota?" com botão Enviar
- [ ] **Tela 3 (obrigado):** Animação de check ✓, "Obrigado pelo feedback! 🚀", texto "Sua resposta foi registrada."
- [ ] Transição suave entre telas (fade ou slide)

### Técnico
- [ ] Token lido de `?token=` na URL query string
- [ ] Busca survey via `supabase-js` com chave anon: `survey_links?token=eq.{token}&select=*,surveys(*)` 
- [ ] Submit via fetch POST para Edge Function `submit-survey`
- [ ] Sem autenticação necessária (anon key)
- [ ] Responsivo: funciona perfeitamente em celular (alunos abrem pelo WhatsApp)
- [ ] Dark premium: usa `design-tokens-dark-premium.css`
- [ ] Sem dependências externas além do Supabase SDK CDN

### Visual (referência: Tally dark)
- [ ] Fundo `#0a0a0a`, fonte Inter
- [ ] Card centralizado com padding generoso
- [ ] Pergunta em destaque (font-size 24-28px, font-weight 700)
- [ ] Botões NPS: grid de 11 botões (0-10), cada um ~44px, com hover e estado selecionado
- [ ] Estrelas CSAT: SVG grandes (~48px), amarelo ao hover/select
- [ ] Botão de avançar: primário com cor da brand (violet #6366f1)
- [ ] Progress bar no topo mostrando progresso (tela 1 de N)
