SESSÃO 1

Business

Edgard Pacheco
Como fazer scraping de dados de viagens sem acesso à API do Booking.com, evitando detecção como spam e problemas jurídicos?
José Carlos Amorim (Zé)
Use MCP (Microsoft Playwright) para simular navegação humana, incluindo cliques em botões de verificação "sou humano". O MCP é indicado especialmente quando não há API disponível. Rode o software localmente na máquina do usuário para reduzir custos e use VPS apenas para acesso via browser.
Betty Bonaparte
Como estruturar o setor de marketing da empresa usando AIOS, desde a pesquisa de APIs até a publicação automatizada em redes sociais?
José Carlos Amorim (Zé)
Passo 1: Pesquise APIs necessárias (Meta, LinkedIn, AppFI). Passo 2: Construa um squad via Squad Creator inserindo as APIs como ferramentas. Passo 3: Crie um "feio que funciona" em 1 semana, focando em fechar o fluxo de criação até postagem no Instagram. O Claude Code fará a simbiose entre APIs e squad automaticamente. PRD e Briefs não são necessários neste estágio inicial.
Bruno (PulsifyAI)
Como usar AIOS para otimizar e acelerar a construção de agentes, código e dashboards que consomem muitas horas de desenvolvimento?
José Carlos Amorim (Zé)
Primeiro passo: Suba todos os projetos visuais (Bolt, Lovable, AI Studio) para o GitHub para corrigir o débito técnico gerado por essas ferramentas, que quebram ao escalar. Esse processo se chama "Brownfield Discovery". Depois, use o AIOS como analista para pesquisar e fazer benchmark, sugerindo melhorias no dashboard. O AIOS guiará a conexão com GitHub e correção do débito técnico.
Cleverson Silva
Como os agentes de viagem rentabilizarão com a plataforma? Há parcerias com operadoras?
Edgard Pacheco (esclarecimento)
A plataforma é para uma empresa de marketing multinível específica (nicho fechado), não um sistema aberto. O objetivo é facilitar a pesquisa que hoje leva 10-15 minutos. O dinheiro já cai automaticamente na conta do agente através do sistema existente da empresa.
Virginia Marcal
Como começar com AIOS tendo que aprender N8N, Supabase e outras tecnologias simultaneamente?
Renato Lhamas
Escolha um projeto vertical específico (exemplo: checklist automatizado para cerimonial) e se lance no processo. Contrate uma VPS barata para hospedar o banco de dados (Supabase usa PostGres) necessário para rodar N8N. Antes de automatizar, tenha o processo desenhado. Siga a documentação passo a passo para instalação do AIOS no Claude Code.
SESSÃO 2

Fundamentos

Gaston (Eduardo K)
Como usar efetivamente o Claude Code e por onde começar quando se está perdido?
João Lozano
Use Claude Code dentro do terminal, onde ele carrega todos os arquivos do projeto para comunicação contextualizada. Comece explorando o sistema conversando com o agente iOS Master. Use o comando /guide para puxar todos os comandos de um agente e suas explicações. Crie um arquivo markdown contextualizando quem você é, seu nível técnico e interesses para os agentes adequarem suas respostas.
Fabiana Furtado
É melhor estudar pelo próprio sistema ou usar notebook LLM com transcrições das aulas?
João Lozano
Use o notebook LLM de forma complementar, não substitutiva. A exploração interna do sistema deve ser o foco inicial. Dúvidas que surgirem podem ser levadas ao notebook LLM para cruzar informações. Subir arquivos do GitHub no notebook (em vez de apenas links) pode causar desatualização, mas os fundamentos do AIOS não mudam significativamente entre versões.
Eduardo K
Por que o comando "clear" não funciona para limpar a tela do terminal?
João Lozano
No AIOS, todos os comandos devem começar com barra (/). O comando correto é /clear. Alternativamente, abra um novo terminal, o que equivale a uma nova instância do projeto. Não se intimide com o terminal - é apenas a interface raiz, similar ao Chat GPT.
Marcelo Matos
O que é o arquivo claude.md e qual sua função no sistema?
João Lozano
O claude.md é o "cérebro central" e "contexto universal" da operação, contendo as regras e leis do projeto. É criado usando o comando /init, que averigua o projeto e gera o arquivo. Cuidado para não sobrescrever as regras do núcleo do AIOS (contexto core), que são fundamentais para o sistema funcionar.
Marcelo Matos
Squads e agentes internos podem ser usados em sistemas em produção?
João Lozano
O AIOS é primariamente um framework de desenvolvimento. Para integrar um squad em produção (como via API), simule o projeto com os agentes e depois use-o. Para transformar um squad em produto com interface para clientes, siga o processo de desenvolvimento do AIOS usando o Meta AIOS. Isso é mais eficiente por limitar o escopo.
SESSÃO 3

Técnico Avançado

Renata Baldissara-Kunnela
O que é o processo de Preflight no Brownfield Discovery e preciso esperar ele terminar?
Talles Souza / Pedagógico Academia Lendária
Preflight é uma simulação (dry run) que testa o ambiente sem fazer alterações - funciona como controle de qualidade (QA) inicial para identificar melhorias. SIM, você deve esperar o processo terminar completamente antes de prosseguir.
Jose Petrucio Da Silva
Por que o Dev está chamando meus agentes pré-existentes (não-AIOS) durante o Brownfield, causando problemas?
Talles Souza
Isso ocorre porque agentes pré-existentes são ativados pelo Claude Code e não pelo AIOS, o que não está previsto no cenário atual. Solução: isole o ambiente, separando os agentes ao usar o AIOS. Use os agentes customizados isoladamente apenas onde for necessário. Alternativamente, crie uma skill que chame o agente Swift customizado, adicionando camada sem alterar estrutura central do Dev.
Jose Petrucio Da Silva
Como transportar um agente especialista (ex: Swift) do Squad Creator para dentro do AIOS corretamente?
Talles Souza / Pedagógico Academia Lendária
Como o AIOS nasceu antes das skills do Claude, esse problema ocorreu. Use o AIOS Master para criar agentes e workflows - ele está preparado para registrar comandos no Claude MD e colocar as peças no lugar certo. Na release atual, não dependa do Squad. Alternativamente, crie uma skill que chame o agente customizado, ao invés de alterar a estrutura original do Dev.
Jose Petrucio Da Silva
Por que o paralelismo de waves gerou 29 subagentes do Dev em paralelo, travando meu Mac de 24GB?
Talles Souza / Pedagógico Academia Lendária
Waves ainda não estão totalmente implementadas na release atual. Cada subprocesso do Claude consome 500 MB de memória, limitando execução de subagentes. Execução simultânea é problemática pois um subagente pode modificar o trabalho do outro. Use Control+E e Control+C para recuperar sessão e analisar logs. Recomendação: abra um AIOS Master paralelo para monitorar o que está acontecendo por trás e controlar melhor a chamada dos agentes.
Marcus Vinicius Coelho
Qual a diferença entre "npx aios-core install" e "npx core init" para projetos clonados?
Pedagógico Academia Lendária / Talles Souza
npx aios-core install: instala o AIOS Core dentro de um projeto EXISTENTE (clonado do GitHub). npx core init: cria um NOVO repositório quando não há projeto no GitHub. O NPX puxa o componente baseado no último release (versão validada), diferentemente do git clone que puxa a versão mais recente da branch main.
Vlademir Fagundes / Renata Baldissara-Kunnela
Posso modificar o AIOS Core open source e usar corporativamente? Posso revender?
Pedagógico Academia Lendária / Talles Souza
SIM, pode modificar (faça fork no Git para não perder alterações e contribuir via pull request). SIM, pode usar corporativamente como ferramenta de trabalho para produzir desenvolvimentos para clientes. NÃO pode revender ou compartilhar o código-fonte do AIOS Core em si, mas pode informar que o utiliza como framework. As regras para o AIOS Pro ainda não foram definidas publicamente e dependerão do modelo de licenciamento.

Insights Valiosos

Filosofia Prática: "Feio que funciona" em 1 semana supera perfeição demorada - validação real > código perfeito
Ferramentas Visuais: Bolt, Lovable e AI Studio são úteis para protótipos mas geram débito técnico crítico ao escalar - GitHub + Brownfield Discovery é essencial
Nichos Fechados: São vantagem estratégica na era da IA - aproveite para criar casos de sucesso antes de mercado abrir
Terminal Desmistificado: Não é inimigo - é interface de conversação contextualizada, similar ao Chat GPT mas com poder de execução
Exploração Ativa: Errar, perguntar, testar supera estudo passivo - sistema recompensa experimentação
Isolamento de Ambientes: Agentes customizados vs AIOS devem ser separados para evitar conflitos imprevistos e economizar horas de debug
Hardware Importa: Paralelismo de subagentes consome 500 MB cada, planeje capacidade antes de escalar
Open Source Estratégico: AIOS Core bem feito acelera adoção corporativa mantendo controle sobre versões premium (AIOS Pro)
Mindset Shift: Usuário é cérebro orquestrador, não executor - fundamental para aproveitar AIOS completamente

Takeaways Práticos
