Resumo Executivo

Nesta aula de 3h10min, Alan Nicolas e Pedro Valerio Lopez desmistificaram a arquitetura completa do AIOS Framework e transformaram a forma como você deve pensar desenvolvimento. Alan começou com um choque de realidade: tokens NÃO são luxo, são infraestrutura. A lição: pare de economizar e foque em CRIAR. Pedro desmontou o AIOS peça por peça - .aios-core, expansion-packs, claude.md. A hierarquia Epic → Story → Task foi o ponto de virada. O momento "mind blowing" da aula: criar DOCUMENTOS estruturados funciona MELHOR que chat conversacional. Essa foi a sacada contra-intuitiva que fez a turma explodir em reações.

Principais Aprendizados

Tokens como Energia Elétrica

R$200/mês não é custo, é infraestrutura essencial (como luz/internet). Alan mostrou que 1 mês do trabalho dele equivale a R$1M em time tradicional - enquanto gasta R$200-500 em tokens.

✅ Assinar Claude Pro ou API esta semana

Estrutura AIOS

.aios-core é o framework base (nunca edite!), expansion-packs são suas customizações, e claude.md é a "Terra" - regras globais. Tudo na mesma pasta compartilhando configuração.

✅ Explorar estrutura nos próximos 3 dias

Epic → Story → Task

Epic = grande objetivo, Story = pedaço entregável, Task = ação específica. Essa decomposição permite AIOS não se perder em contexto gigante.

✅ Praticar decomposição no papel

@ vs / - Diferença Crítica

@ = persona-switching (apenas muda tom), / = spawn agente DE VERDADE (executa workflow completo com ferramentas). Essa distinção é crucial para usar AIOS corretamente.

✅ Usar / para execução real

Ferramentas Essenciais

CodeRabbit (code review IA), GitHub Actions (CI/CD automatizado), Environment Bootstrap (instala tudo automaticamente). Cada ferramenta tem papel específico no workflow.

✅ Criar conta CodeRabbit gratuita

Documentação > Conversa

Revelação contra-intuitiva: criar DOCUMENTOS estruturados funciona MELHOR que chat conversacional. Documentos dão contexto claro e determinismo. Chats levam a ambiguidade.

✅ Criar brief.md ANTES de chamar AIOS

BOB (Baby On Board)

Agente para iniciantes com modo "rodinhas" que guia você passo a passo. Não tenha vergonha de usar BOB - até experts usam quando estão aprendendo nova área.

✅ Usar /bob para próximo projeto pequeno

Monorepo

Tudo na mesma pasta - múltiplos projetos compartilhando configuração. Simplifica gerenciamento e mantém consistência entre projetos.

✅ Entender estrutura monorepo

Ferramentas Essenciais

CodeRabbit

Essencial

Ferramenta de code review com IA que integra diretamente com GitHub. Analisa pull requests automaticamente e fornece feedback técnico detalhado.

GitHub Actions

Essencial

CI/CD automatizado (Continuous Integration / Continuous Deployment). Executa testes, build e deploy automaticamente a cada commit.

Environment Bootstrap

Essencial

Comando /devops *environment-bootstrap que instala TUDO automaticamente - dependências, configurações, ferramentas.
